# data/strategy — 世界杯下注策略脚本集

围绕"**用 2022 全程 + 2026 在赛数据,找可重复的下注 edge**"做的一整套验证 + 出单工具。
核心不是"赔率内部套利"(体彩定价高度自洽、套不出),而是**认知筛选 + 本届择时 + 串关杠杆**这条闭环。

## 一句话打法闭环

```
筛选(原型/赔差 → 降下场率,只打信号明确的)
  → 本届指纹(这一届什么脾气 → 决定方向/腿是否 +EV)
  → 出单(单注为主;比分仅辅助收口)
  → 串关(只把 +EV 腿 clean 杠杆放大;爆冷年改用覆盖胆对冲)
```

**关键认知**:串关/覆盖**不创造 edge**,只放大或对冲单腿的 EV;edge 来自"选对场(原型)+ 选对年景(指纹)"。看不懂的场不下场。

## 数据依赖(脚本只读,不写业务数据)

- `data/2022wc/id_map.json` + `odds/` + `results/` —— 2022 卡塔尔(全程,含赛果)
- `data/settled_matches.json` —— 2026 已完赛(spf/rqspf/bf/zjq/bqc 的 initial/last + result)
- `data/matches.json` —— 2026 赛程(组别/日期 → 推轮次)
- `data/odds/<mid>.json` —— 2026 各场最新赔率
- `modeling/artifacts/predict_31_<日期>.json` —— 31 规则出单(daily_dual_report 读它)

## 生产脚本(日常就跑这两个)

### `daily_dual_report.py` ⭐ 出单主入口
- **说明**:读最新 `predict_31_*.json` + 赔率 + R1 积分 + 本届指纹,生成"**31规则出单 + 新策略出单**"合并 markdown,写入 `modeling/artifacts/出单合并_<日期>.md`。
- **用法**:`python3 data/strategy/daily_dual_report.py 2026-06-20`
- **结论/产出**:每场给原型 + E[球] + 积分 + 信心(★/中/⚠/✗)+ 方向 + 比分 + 串关建议。可直接挂进 `wc2026-daily` Step 5 之后。

### `tournament_tracker.py` ⭐ 本届指纹滚动追踪
- **说明**:读当前已赛,输出本届指纹(热门/冷门/平局 兑现 vs 隐含、场均球)、**环境拆分**(高海拔/高温/温和球场)、下一轮顺势倾向、前瞻自动记分。每轮赛后重跑。
- **用法**:`python3 data/strategy/tournament_tracker.py`
- **结论**:2026 = **高球年(场均 3.05)+ 和棋多(+11%)+ 冷门不灵(−7%)**,与 2022(低球爆冷年)相反;**高海拔/高温球场进球更多**(3.5/3.38 vs 温和 2.70),印证"举办地气候影响赛果"。

## 选择层 / 原型

### `match_archetypes.py` 比赛原型分类器(认知筛选总闸)
- **说明**:按 **强弱轴(spf 热门赔率)× 进球轴(zjq 隐含均球)** 分 9 原型,给每类可读度 + 押热/平/冷 ROI + Top 实际比分。
- **用法**:`python3 data/strategy/match_archetypes.py`
- **结论**:★**强打弱+大球型**→押热门(75% 兑现);**中等+中球**→偏平(押平 +31%);⚠**强打弱+闷局(中球)**=反直觉**爆冷温床**(29% 冷、押热 −33%);**均势局=看不懂→跳过**。

## 机制证据层(为什么这么打)

### `points_mentality_2022.py` 出线心态错价
- **说明**:重建 2022 各轮积分,按赛前出线形势(养生/搏命/双方同分)分桶看押冷/押平 ROI。
- **结论**:**冷门红利集中在 R2/R3**(押冷 R1 −16% → R2 +31% → R3 +134%);**养生局**(一方已稳 vs 一方需分)押其对手 **+207%**;**R3 平局崩塌**(押平 −79%)。

### `rqspf_spread.py` 让球高低赔差筛选 + 覆盖
- **说明**:rqspf 高低赔差按阈值筛,看押热/冷/平 + "热+平" + "高+低两边" 的命中与 ROI。
- **结论**:高赔差 = 高可读(≥3.0 时结果一边倒、下场率 17%),但**押热 +EV 随年景翻转**(2022 +25% / 2026 −23%);**押冷处处负**(让球盘内部也有热门-冷门偏差);覆盖只降波动、封顶 ≈ −6%,不提 ROI。

### `fade_public_by_round.py` 反向去除大众侧 × 心态
- **说明**:fade 名门/最低赔热门(大众侧),按轮次 + 大众侧赛前积分分桶。
- **结论**:**跟买大众侧两届稳亏**(被超买);但"反买对手"只在 **R2/R3 养生局**转正(+201%),**R1 反买最惨(−51%)**。fade = 排除大众侧,不是无脑反买。

### `bf_exclusion_strategy.py` 比分排除法
- **说明**:先判爆冷不(3 种 gate:从不/均势/oracle),爆冷押弱队小比分、不爆冷反市场热度取中赔,排除法选分回测。
- **结论**:bf 高水位(全押 −54%);**只在 WEAK 局 + 押最便宜低分簇** 最优但仍 −15%;大球信号该用 **zjq(总进球)** 表达,别押精确比分(猜格税)。

## 串关层(只放大 +EV,不创造 edge)

### `rqspf_banker_parlay.py` 串关=杠杆
- **结论**:串关把各腿 (1+ROI) **相乘**。正 EV 腿越串越肥(2026 让球热门 +34% → 3串1 +194%);负 EV 腿越串越亏(2022 −13% → −67%)。**先确认腿 +EV,再串**。

### `rqspf_cover_parlay.py` 覆盖胆 = 条件性对冲
- **结论**:把"高+低两边"覆盖腿当胆塞进串关,**抬命中率但是对冲不是放大**:爆冷年抬 ROI(2022 clean −43% → 覆盖 +17%),热门年稀释(2026 clean +92% → 覆盖 −4%)。年景定 clean / 覆盖。

### `prob_parlay.py` 串高概率 vs 串高赔
- **结论**:**只有 ≤1.3 超强热门是 +EV**(+8% / 命中 89%,热门-冷门偏差);串高概率(正 EV 腿)> 串高赔(冷门负 EV 腿);但铁腿稀缺(65 场仅 9 场)、爆冷年也会塌 → 低频精选、需择时。

## 诚实刹车(看任何结论前先读)

- 样本小:**2022 全程 45 场 + 2026 在赛 20–28 场**;原型最准的格子也才 n≈16–21。
- **ROI 幅度普遍不可信**(常被 1–3 场大冷顶起),**方向 + 机制可信**。
- "信热门 / 爆冷年"是**进行中的读数**,会随 2026 的 R2/R3 变脸 → 串关仓位保守。
- 体彩 5 玩法定价高度自洽(spf vs bf 隐含仅差 ~1.9%),**没有跨市场套利**;edge 只在选择 + 择时。
- 比分(bf)是最高水位的载体,主力价值在**方向(spf/rqspf)与总进球(zjq)**。
