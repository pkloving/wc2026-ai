关于这个站点

🎯 目的

这是一个静态站点，把 2026 美加墨世界杯的赛程、比分，跟我事先问各大 AI 大模型的预测记录放在一起对比。世界杯期间，每场比赛结束后我都会来更新比分和命中情况。

🧪 实验设计

  1. 每场比赛开赛前 24 小时内，用同一个 prompt（见每场 AI 预测页）问几个大模型。
  2. 比赛结束后，更新比分 → 系统自动判定每个模型是 ✅ 比分命中 / ⚠️ 胜负命中 / ❌ 未中。
  3. 所有比赛结束后，统计页会给出每个模型的最终战绩。

🤖 对比的 AI 模型

暂定：MiniMax-M3、Claude Opus 4.8、GPT-5 mini、DeepSeek-V4、Kimi。

📊 命中判定规则

  • 比分命中：预测比分 == 实际比分（包括 90 分钟比分，淘汰赛不算加时/点球）
  • 胜负命中：胜平负方向一致，但比分不完全一致
  • 未中：胜平负方向不一致

🛠 技术栈

  • Vite + 原生 HTML/CSS/JS（多页面）
  • Tailwind CSS
  • Chart.js（统计页图表）
  • 纯静态、可部署到 GitHub Pages / Vercel / Netlify

📁 项目结构

wc2026-ai/
├─ index.html / schedule.html / standings.html / results.html
├─ predictions.html / stats.html / teams.html / about.html
├─ match.html?id=Mxxx  (单场详情)
├─ data/             # matches / results / predictions / teams / groups
├─ public/data/      # 同上，作为 Vite 静态资源

├─ css/main.css      # Tailwind 入口
└─ scripts/          # generate-matches / update-result / add-prediction
