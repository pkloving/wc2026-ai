🎫 给用户充值的两种方式

方式 A（直接充值，推荐）：在下方「快速充值」填邮箱+积分数→点「直接充值」，积分秒到账，无需用户操作。

方式 B（License Key）：在下方「License Key 发放」生成 key → 复制发给用户 → 用户在 chatbot 点「兑换 key」。

区别：直接充值适合熟人/深度用户；License Key 适合公开售卖（用户自己输 key，更正式）。

🤖 模型预测数据怎么更新

注：聊天框「出今日推荐」按钮已下线（定位转向「卖研究/工具，不卖推荐」）。以下流程仍用于刷新 simulate / predictions 展示的模型预测数据。

Vercel 上不跑 modeling（serverless 限制），需要你本地生成后 push。

每日流程：

  1. node modeling/scripts/31_tight_anti_value.js <YYYY-MM-DD> --predict
  2. node scripts/build_chat_predict.js <YYYY-MM-DD>
  3. git add && git commit && git push

预测数据通常在 17:00 左右赔率收敛后生成。如果当天没有未完赛比赛，文件可能为空。
